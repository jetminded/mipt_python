# Web App Project

Проект содержит несколько Streamlit приложений для демонстрации различных возможностей фреймворка.

## Приложения

### 1. simple_app/simple_web_app.py
Простое веб-приложение, демонстрирующее базовую интерактивность:
- Поле для ввода имени пользователя
- Кнопку отправки формы
- Приветственный баннер с именем пользователя
- Анимацию воздушных шариков при успешной отправке

### 2. web_app.py
Основное многостраничное приложение с навигацией, включающее:
- **Cowsay** - страница с генерацией ASCII-арт коровы
- **Maps** - страница с картами

## Установка и запуск

### Предварительные требования
- Python 3.8+
- uv (установщик пакетов)

### Установка зависимостей
```bash
uv sync
```

### Запуск приложений

#### Основное приложение
```bash
# Простое приложение
uv run streamlit run simple_app/simple_web_app.py

# Основное приложение
uv run streamlit run web_app.py
```

После запуска приложения будут доступны по адресу: http://localhost:8501
