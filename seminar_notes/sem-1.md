# Руководство по настройке среды разработки Python

## 1. Установка базовых инструментов

### 1.1 Установка Python

Python — основной язык программирования курса. Нам нужна версия 3.8 или выше.

#### Для всех операционных систем:
1. Перейдите на официальный сайт: https://www.python.org/downloads/
2. Скачайте последнюю стабильную версию Python (рекомендуется 3.11 или 3.12)
3. Запустите установщик

#### Важные настройки при установке:

**Windows:**
- Обязательно отметьте "Add Python to PATH"
- Выберите "Install for all users" (если есть права администратора)

**macOS:**
- Установщик автоматически настроит PATH
- Если у вас уже установлен Python через Homebrew, можете использовать его

**Linux (Ubuntu/Debian):**
```bash
sudo apt update
sudo apt install python3 python3-pip python3-venv
```

#### Проверка установки:
Откройте терминал (Terminal на macOS, PowerShell на Windows, Terminal на Linux) и выполните:

```bash
python --version
# или если не работает:
python3 --version
```

**Ожидаемый результат:** `Python 3.x.x` (где x.x — номер версии)

### 1.2 Установка Git

Git необходим для загрузки материалов курса и работы с репозиторием. Если у вас его нет, установите git по [мануалу от GitHub](https://github.com/git-guides/install-git)

#### Проверка установки:
```bash
git --version
```

**Ожидаемый результат:** `git version 2.x.x`

## 2. Получение материалов курса

### 2.1 Клонирование репозитория

Откройте терминал и выполните следующие команды:

```bash
# Переходим в папку Desktop (или выберите другую удобную папку)
cd ~/Desktop

# Клонируем репозиторий курса
git clone https://github.com/jetminded/mipt_python.git

# Переходим в папку с материалами
cd mipt_python
```

#### Проверка:
Убедитесь, что папка создалась и содержит файлы курса:

```bash
# Посмотрим содержимое папки
ls -la
# На Windows используйте: dir
```

**Ожидаемый результат:** Вы должны увидеть папки `base/`, `adv/`, `hw-adv/` и файлы `README.md`, `sem-1.md`.

Обновляйте локальную версию репозитория по мере того, как мы заливаем новый материал, командой `git pull`

### 2.2 Альтернативный способ (если Git не работает)

Если возникли проблемы с Git:
1. Перейдите на https://github.com/jetminded/mipt_python
2. Нажмите зеленую кнопку "Code" → "Download ZIP"
3. Распакуйте архив в удобную папку
4. Переименуйте папку в `mipt_python`


### 2.3 [Опционально] Google Colab (без установки Python)

**Если у вас возникли сложности с установкой Python или вы хотите быстро начать работу**, можете использовать Google Colab:

1. После клонирования репозитория найдите файлы с расширением `.ipynb` в папке `base/`
2. Перейдите на https://colab.research.google.com/
3. Выберите "Upload" и загрузите нужный `.ipynb` файл
4. Google Colab предоставляет готовую среду Python с предустановленными библиотеками

**Преимущества Google Colab:**
- Не требует установки Python на компьютер
- Предустановленные популярные библиотеки (numpy, pandas, matplotlib)
- Бесплатный доступ к GPU для вычислений
- Автоматическое сохранение в Google Drive

## 3. Настройка виртуального окружения

Виртуальное окружение изолирует зависимости проекта от системного Python, предотвращая конфликты между пакетами.

### 3.1 Создание виртуального окружения

Убедитесь, что вы находитесь в папке `mipt_python`:

```bash
pwd  # Проверяем текущую папку
# Должно показать путь, заканчивающийся на mipt_python
```

#### Для Windows:
```bash
# Создаем виртуальное окружение
python -m venv .venv

# Активируем его
.venv\Scripts\activate
```

#### Для macOS/Linux:
```bash
# Создаем виртуальное окружение
python3 -m venv .venv

# Активируем его
source .venv/bin/activate
```

### 3.2 Проверка активации

После активации в начале строки терминала должно появиться `(.venv)`:

```bash
(.venv) user@computer:~/Desktop/mipt_python$
```

#### Проверка Python в виртуальном окружении:
```bash
which python  # macOS/Linux
where python  # Windows
```

Путь должен указывать на Python внутри папки `.venv`.

## 4. Установка Jupyter Notebook

### 4.1 Установка пакетов

С активированным виртуальным окружением выполните:

```bash
# Обновляем pip до последней версии
python -m pip install --upgrade pip

# Устанавливаем Jupyter Notebook
python -m pip install notebook

# Устанавливаем дополнительные полезные пакеты
python -m pip install numpy pandas matplotlib
```

### 4.2 Проверка установки

```bash
jupyter --version
```

**Ожидаемый результат:** Информация о версиях Jupyter компонентов.

---

## 5. Запуск и проверка работы

### 5.1 Запуск Jupyter Notebook

```bash
# Убедитесь, что виртуальное окружение активировано
# В начале строки должно быть (.venv)

jupyter notebook
```

**Что произойдет:**
1. В терминале появятся сообщения о запуске сервера
2. Автоматически откроется браузер с интерфейсом Jupyter
3. Вы увидите файловый менеджер с содержимым папки `mipt_python`

### 5.2 Тестирование

1. Откройте файл `base/01_base_syntax.ipynb`
2. Попробуйте выполнить первую ячейку (Shift + Enter)
3. Если код выполняется без ошибок — настройка прошла успешно!

### 5.3 Завершение работы

Для остановки Jupyter Notebook:
1. Закройте вкладки браузера
2. В терминале нажмите `Ctrl + C`
3. Подтвердите остановку, нажав `y`

Для деактивации виртуального окружения:
```bash
deactivate
```

## 7. [Дополнительно] Настройка VS Code

Visual Studio Code — мощная IDE, которая может улучшить ваш опыт разработки на Python.

### 7.1 Установка VS Code

1. Скачайте с официального сайта: https://code.visualstudio.com/
2. Установите, следуя инструкциям для вашей ОС

### 7.2 Установка расширений

Откройте VS Code и установите следующие расширения:

**Обязательные:**
- **Python** (Microsoft) — основная поддержка Python
- **Jupyter** (Microsoft) — работа с .ipynb файлами

### 7.3 Настройка интерпретатора Python

1. Откройте папку `mipt_python` в VS Code: `File` → `Open Folder`
2. Откройте Command Palette: `Ctrl+Shift+P` (Windows/Linux) или `Cmd+Shift+P` (macOS)
3. Введите и выберите: `Python: Select Interpreter`
4. Выберите интерпретатор из виртуального окружения: `.venv/bin/python` (macOS/Linux) или `.venv\Scripts\python.exe` (Windows)

### 7.4 Работа с Jupyter в VS Code

1. Откройте любой `.ipynb` файл
2. VS Code автоматически предложит установить необходимые расширения
3. Выберите правильный Python интерпретатор (из .venv)
4. Теперь можете выполнять ячейки прямо в VS Code!
